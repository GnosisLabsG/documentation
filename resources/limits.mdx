## Supported file types

| Format | Extension | Max file size | Max content |
|--------|-----------|--------------|-------------|
| PDF | `.pdf` | 50 MB | 500 pages |
| Word | `.docx` | 50 MB | 300 pages |
| Text | `.txt` | 10 MB | 5,000,000 characters |
| Markdown | `.md` | 10 MB | 5,000,000 characters |

### Chat image uploads

| Property | Limit |
|----------|-------|
| Max images per message | 4 |
| Max file size per image | 20 MB |
| Supported formats | JPEG, PNG, GIF, WebP, BMP, TIFF |

### Booking form file uploads

Supported types: PDF, DOCX, XLSX, PPTX, PNG, JPG, TXT

## Scenario validation

| Rule | Requirement |
|------|-------------|
| Start nodes | Exactly 1 |
| End nodes | At least 1 |
| Keyword trigger | At least 1 keyword |
| Choice question | At least 2 options |
| Form node | At least 1 field |

## Booking limits

| Rule | Configurable |
|------|-------------|
| Slot interval | 15, 30, or 60 minutes |
| Max crawl depth | 1–3 levels |
| Max crawl pages | 1–100 pages |
